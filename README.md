# FIFA World Cup Data Analysis Dashboard (1930–2014)

## Project Overview

This project analyzes historical FIFA World Cup data using Tableau to identify trends in team performance, hosting patterns, and tournament statistics.

The dashboard provides interactive insights into World Cup winners, hosting countries, team participation, and competition statistics.

## Objectives

* Analyze the most successful World Cup teams.
* Identify countries that hosted the tournament.
* Explore trends in matches played, goals scored, and qualified teams.
* Visualize historical World Cup results.

## Dashboard Features

* Countries that won the FIFA World Cup
* Teams by total Finals and Semi-Finals
* Countries that hosted the World Cup
* Competition statistics by year
* Countries qualified for the tournament
* World Cup results with stadium details

## Tools Used

* Tableau
* Data Cleaning
* Data Visualization

## Key Insights

* Brazil has the highest number of World Cup titles.
* Italy and Germany consistently reach finals and semi-finals.
* European and South American teams dominate historically.
* Tournament scale has increased over the decades.
